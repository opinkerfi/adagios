About Adagios

Adagios is a web based Nagios configuration interface built to be simple and intuitive in design, exposing less of the clutter under the hood in Nagios.

This is a tech preview and there are a lot of known bugs. We would really like your feedback to help us develop this app.

Features

  • Edit Nagios configs in-place without a database backend (thanks pynag)
  • Audit of configuration changes (aka git integration)
  • Bulk edit/copy/delete objects
  • Value-add for both novices and Nagios gurus alike
  • REST interface
  • Start/stop/reload Nagios service
Adagios also integrates nicely with okconfig offering features such as:
  • Template support
  • Preconfigured templates (Windows, Linux, Oracle, Brocade, and more...)
  • Remotely push Nagios agent to clients (Windows, RHEL, Debian and SUSE are supported)
  • Network scan to find unmonitored hosts
  • Easy changes to plugin thresholds via Nagios macros
